# Changelog

## [1.0.0] - 2025-12-26

### Added

- Initial release of UF to CLP Converter
- Convert between UF (Unidad de Fomento) and CLP (Chilean Peso)
- Real-time UF value fetching from mindicador.cl API
- Daily caching of UF values (24-hour cache)
- Smart input parsing with automatic direction detection
- Copy to clipboard functionality (press Enter on results)
- Display of current UF value and date
- Refresh action to reload UF value from API
